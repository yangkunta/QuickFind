# 變更紀錄 (Changelog)

## [v1.1.15] - 2026-07-11
### 新增 (Features)
- **清單顯示優化**：在查詢與維護模式的清單中，將資料的「分類」以藍色小標籤的方式顯示在備註內容的前方，讓資料辨識更加直覺。

## [v1.1.14] - 2026-07-11
### 修復 (Fixes)
- **刪除紀錄提示**：修正刪除某筆紀錄時，確認對話框會顯示 `undefined` 的問題，現在會正確顯示紀錄的備註內容前 20 字或分類名稱。

## [v1.1.13] - 2026-07-11
### 新增 (Features)
- **匯出功能強化**：點擊「匯出加密檔案 (JSON)」後，現在會直接將檔案靜默儲存到手機的 `Documents` (文件) 目錄中，不再彈出容易造成混淆的「分享」畫面，並加入成功提示。

## [v1.1.12] - 2026-07-10
### 修復 (Fixes)
- **APK 打包問題**：修復 GitHub Actions 打包 APK 時缺少 `@capacitor/filesystem` 與 `@capacitor/share` 模組的問題，確保 Android 原生儲存功能正常運作。

## [v1.1.10 ~ v1.1.11] - 2026-07-10
### 變更 (Changes)
- **雲端備份移除**：依據需求，將 Google Drive 雲端同步相關功能與設定介面完全移除。
- **OTA 更新邏輯修正**：修復 GitHub Pages 上的 OTA 版本無法正確更新到手機的問題。
- **UI 體驗優化**：
  - 將「設定」畫面的標題字體改小並設定為紫色。
  - 將最下方的主功能導覽列 (Bottom Navigation Bar) 固定在螢幕底部，確保在滑動內容時導覽列不會跟著捲動或被遮蔽。
  - 修復內容區塊捲動失效的問題。
- **防呆機制**：若使用者安裝了舊版的 APK (缺少原生 Capacitor 擴充模組)，在執行需要底層權限的操作 (如匯出) 時會清楚跳出 Toast 提示「請至 GitHub 下載最新版 APK」。
