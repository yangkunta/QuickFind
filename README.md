# QuickFind (記憶安全金庫)

QuickFind 是一個極致簡化、高安全性的本地端私密筆記管理器。
這個專案專注於提供**最快、最安全的查詢與維護體驗**，完全在本地端加密，確保資料零外洩。

## 🎯 產品目的
- https://qfmyp.netlify.app/ (發佈到 Netlify 只做OTA時使用)
- 提供一個**沒有學習門檻**的密碼與機密資料管理介面。
- 確保所有資料都在使用者設備端加密 (Zero-Knowledge Architecture)，不經過任何後端資料庫。
- 專為手機 (行動端) 操作優化的 OTA 熱更新架構，版更時不用每次更新APK。
- 提供大批資料的匯入 (CSV) 以及本地/雲端備份機制，確保資料永不遺失。

## 🛠️ 技術架構

- **前端核心**：純 HTML5, CSS3, Vanilla JavaScript (無框架，極致輕量化)。
- **安全加密**：使用瀏覽器原生的 `Web Crypto API` (AES-GCM 256-bit 演算法與 PBKDF2 金鑰導出)。
- **資料儲存**：`localStorage` (加密後儲存)，完全去中心化。

## 🚀 核心功能說明

### 1. 極簡管理介面
- 移除傳統密碼管理器冗長的欄位，只保留最核心的：「分類」、「備註(多行)」、「說明(多行)」。
- 智慧圖示：會根據您輸入的分類或備註關鍵字（如：卡、登入、mail、銀行），自動顯示相應的圖示。

### 2. 即時查詢與維護模式
- **查詢模式**：預設為安全空白狀態。輸入關鍵字時提供「邊打邊找」的即時過濾功能；列表僅提供「一鍵複製」，防止誤改資料。
- **維護模式**：專屬的維護頁面，每筆資料顯示「編輯」與「刪除」按鈕，方便您整批管理資料。
- **顯示全部**：提供一鍵顯示所有資料的功能。

### 3. 高安全性身分驗證
- **主密碼保護**：進入保險箱前必須輸入主密碼進行解鎖。
- **敏感操作驗證**：在進行「匯入」、「還原」、「修改主密碼」等敏感操作時，皆會再次要求驗證主密碼。
- **自動上鎖**：待機一段時間後系統會自動鎖定，保護隱私。

### 4. 完整的資料匯入與備份
- **CSV 整批匯入預覽**：支援從 CSV (三欄格式：分類, 備註, 說明) 批量建檔，匯入前提供**視覺化預覽**與勾選介面。
- **本地端備份/還原**：可將加密後的金庫匯出為 `.vault` 或 `.json` 檔，並在其他設備還原。
